# Swetam Meher Portfolio

Static portfolio website for Swetam Meher, Software Engineer II and ETL Developer.

## Sections

- Interactive ETL pipeline lab
- Case studies and work themes
- Experience timeline
- Skills and capabilities
- Resume download
- Contact links

## Run Locally

```bash
python3 -m http.server 4174
```

Then open `http://127.0.0.1:4174/`.

## Deploy

This is a static site and can be hosted on GitHub Pages, Netlify, Vercel, or Cloudflare Pages.
